Este projeto está sendo desenvolvido pela **CodEX Academy**, um braço da CodEX focado na aceleração de novos desenvolvedores. O objetivo é entregar uma Landing Page moderna, responsiva e performática para a **Associação Atlética Acadêmica Octabit (UNIFEV)**.

##  Estrutura de Arquivos
Para manter a organização, seguimos este padrão:

```text
CODEX-OCTABIT-LP/
├── assets/
│   ├── css/
│   │   ├── style.css         # Estilos globais e layout
│   │   └── variables.css     # Paleta de cores e tipografia
│   ├── img/
│   │   └── icons/            # Ícones e assets de imagem
│   └── js/
│       └── main.js           # Scripts de interatividade
├── .gitignore                # Arquivos ignorados pelo Git
├── CONTRIBUTING.md           # Guia de contribuição da CodEX
├── index.html                # Página principal (Entry point)
├── LICENSE                   # Licença do projeto
└── README.md                 # Documentação principal
```
